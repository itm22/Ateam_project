# 1. 프로젝트 개요

> 성인 ADHD 습관 형성 코치는 지속적인 습관을 형성하기 어려워하는 성인 ADHD 환자들에게 도움을 주는 서비스입니다

# 2. 문제 정의 및 솔루션

## 핵심 문제

- 성인 ADHD 환자는 **집중력 부족, 시간 감각 왜곡, 충동성** 때문에 일상 루틴 유지가 어렵다.
- 기존 습관 형성 앱(Habitica, Fabulous 등)은 일반 사용자 중심 설계 → ADHD 특화 UX와 적응형 피드백 부족
- 이로 인해 **습관 형성 실패 → 낮은 자존감 → 앱 이탈**로 이어지는 악순환 발생

## 솔루션 가설

**"AI 기반 맞춤 알람 시스템과 피드백 기능으로 유저들에게 성취감과 자기 통제감을 향상시킬 수 있다.”** 

# 3. 프론트엔드

## 기술 스택: React

### 역할

- 데이터를 실시간으로 갱신
- 데이터 및 LLM 시각화
- 앱으로 확장

# 4. 백엔드 겸 데이터베이스

## 기술 스택: supabase

### 역할

- 데이터베이스로서 사용자 데이터 관리
- 인증 기능(구글 로그인, 메일 로그인)
- 스토리지로서 프로필 이미지 및 보상을 저장

# 5. 구현 중인 프로젝트 현황

## 구현된 주요기능

### 유저 기본 정보 관리

기능명: 사용자 정보 관리
목적: 사용자의 정보 저장, 기능과 연결
설명: supabase 테이블에 저장

### 사용자 인증

기능명: Google 로그인 / 메일 로그인
목적: 사용자별 루틴 및 보상 데이터 관리
설명: Google Cloud Console에서 OAuth 클라이언트를 생성하여 Supabase Auth와 연동 / Supabase Auth와 연동하여 Magic Link로 이메일 인증 구현
